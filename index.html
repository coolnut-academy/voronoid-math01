ขั้นตอนการใช้งาน:

  1. เลือกโหมด: กรอกเอง หรือ ไฟล์ CSV
  2. เพิ่มตำแหน่งหมู่บ้านตามโหมดที่เลือก
  3. กำหนดจำนวน Sensor Gateway ที่ต้องการติดตั้ง
  4. คลิก "คำนวณตำแหน่ง Gateway"
  5. ระบบจะหาจุดศูนย์กลาง (Hub) ที่ครอบคลุมหมู่บ้านได้ดีที่สุด

  7. ตรวจสอบสถิติ: ระยะทางสูงสุดและเฉลี่ย
  8. ส่งออกผลลัพธ์เป็นไฟล์ CSV หากต้องการ

Sensor Gateway คืออะไร?

คือสถานีแม่ข่าย (Hub) ที่ทำหน้าที่รวบรวมสัญญาณจาก Sensor รอบข้าง โดยจะติดตั้ง ณ จุดกึ่งกลาง (Centroid) ของกลุ่มหมู่บ้านที่เป็น Family เดียวกัน

เกี่ยวกับอัลกอริธึม:

เครื่องมือนี้ใช้อัลกอริธึม k-means / Centroidal Voronoi Tessellation เพื่อหาตำแหน่งติดตั้ง Gateway (Hub) ที่จุดศูนย์ถ่วงของกลุ่มหมู่บ้าน:

  • จัดกลุ่มหมู่บ้าน (Clustering) ให้กับ Gateway ที่ใกล้ที่สุด
  • ย้าย Gateway ไปยัง "จุดกึ่งกลางทางภูมิศาสตร์" ของกลุ่มนั้นๆ
  • ทำซ้ำจนกว่าตำแหน่งจะสมดุล

อัลกอริธึมนี้ให้ผลลัพธ์ที่ดีในภาพรวม แทนที่จะเน้นลดระยะทางไกลที่สุดเท่านั้น

คำแนะนำการใช้งาน:

  • จำนวน Gateway ควรน้อยกว่าจำนวนหมู่บ้าน
  • หากต้องการเพิ่มหมู่บ้านทีละเยอะๆ แนะนำให้ใช้โหมด CSV
  • กด "ดาวน์โหลดตัวอย่างไฟล์" ในแท็บ CSV เพื่อดูรูปแบบที่ถูกต้อง

ฟีเจอร์:

  • อัลกอริธึม k-means: หาตำแหน่งที่เหมาะสมด้วยวิธี centroidal Voronoi
  • การลู่เข้าแบบอัตโนมัติ: หยุดเมื่อเซ็นเซอร์ไม่เคลื่อนที่มาก
  • การแสดงผล Voronoi: แสดงพื้นที่ครอบคลุมของแต่ละเซ็นเซอร์
  • สถิติการคำนวณ: แสดงระยะทางสูงสุดและเฉลี่ย

การแก้ไขปัญหาที่พบบ่อย:

  • ไม่มีผลลัพธ์: ตรวจสอบว่ามีหมู่บ้านอย่างน้อย 1 แห่ง
  • เซ็นเซอร์มากเกิน: จำนวนเซ็นเซอร์ควรน้อยกว่าหรือเท่ากับจำนวนหมู่บ้าน
  • แผนที่ไม่แสดง: กดปุ่ม "แสดงทั้งหมด" เพื่อปรับมุมมอง
  • อัปโหลด CSV ไม่ได้: ตรวจสอบรูปแบบไฟล์และชื่อคอลัมน์
  • Voronoi ไม่แสดง: กดปุ่มแสดง/ซ่อน Voronoi เพื่อเปิดการแสดงผล
